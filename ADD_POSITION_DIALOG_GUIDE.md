# 添加位置对话框 - 功能说明

## ✅ 功能概述

现在 **PositionSettingsView** 的"添加位置"功能已升级为**专业对话框模式**，提供更直观、更强大的位置添加体验！

## 🎯 新增功能特性

### 1. 📋 设备选择器
- **智能设备列表** - 自动加载所有支持位置的设备
- **清晰显示** - 显示设备名称和设备类型
- **分类显示** - CAN电机、EtherCAT电机、离心机、机器人

### 2. 📝 位置参数输入
- **位置名称** - 自定义位置名称
- **位置值** - 精确输入位置值（mm / 度）
- **速度** - 设置运动速度（mm/s / 度/s）
- **实时验证** - 输入验证，防止错误

### 3. ⚡ 快速预设
- **一键预设** - 6种常用位置名称预设
- **智能默认值** - 每种预设自动设置合理的位置值和速度

| 预设名称 | 位置名称 | 默认位置值 | 默认速度 |
|---------|---------|-----------|---------|
| 原点位置 | HOME_POS | 0 | 50 |
| 取样位置 | SAMPLE_POS | 100 | 200 |
| 清洗位置 | WASH_POS | 200 | 150 |
| 等待位置 | WAIT_POS | 0 | 100 |
| 停靠位置 | PARK_POS | 300 | 100 |
| 维护位置 | MAINTENANCE_POS | 500 | 50 |

### 4. 🔄 连续添加模式
- **批量添加** - 勾选"添加后继续添加"
- **快速录入** - 不关闭对话框，连续添加多个位置
- **自动清空** - 每次添加后自动清空表单，保留设备选择

## 📖 使用方法

### 方式一：基本添加（单个位置）

1. 点击 **"添加位置"** 按钮
2. 在对话框中：
   - 选择目标设备（默认选中第一个）
   - 输入位置名称（如 "取样位置"）
   - 输入位置值（如 100）
   - 输入速度（如 200）
3. 点击 **"添加位置"** 按钮
4. ✅ 位置添加成功，自动关闭对话框

### 方式二：快速预设添加

1. 点击 **"添加位置"** 按钮
2. 选择目标设备
3. 点击任一**快速预设按钮**（如"取样位置"）
   - 自动填入位置名称：`SAMPLE_POS`
   - 自动设置位置值：`100`
   - 自动设置速度：`200`
4. 可根据需要微调参数
5. 点击 **"添加位置"** 按钮
6. ✅ 完成

### 方式三：批量连续添加

1. 点击 **"添加位置"** 按钮
2. **勾选**"添加后继续添加"
3. 选择设备，输入第一个位置参数
4. 点击 **"添加位置"**
   - 对话框保持打开
   - 自动清空位置名称、位置值、速度
   - 设备选择保留
5. 继续输入下一个位置参数
6. 重复添加，直到添加完所有位置
7. **取消勾选**"添加后继续添加"或点击"取消"
8. ✅ 完成批量添加

## ⚙️ 预设位置详解

### HOME_POS - 原点位置
- **用途：** 设备的零点/原点
- **默认值：** 位置=0, 速度=50
- **适用：** 所有电机设备
- **说明：** 低速移动到安全原点

### SAMPLE_POS - 取样位置
- **用途：** 样品取样点
- **默认值：** 位置=100, 速度=200
- **适用：** 取样臂、吸液设备
- **说明：** 中高速移动到取样点

### WASH_POS - 清洗位置
- **用途：** 设备清洗点
- **默认值：** 位置=200, 速度=150
- **适用：** 液体处理设备
- **说明：** 中速移动到清洗区域

### WAIT_POS - 等待位置
- **用途：** 设备待机点
- **默认值：** 位置=0, 速度=100
- **适用：** 所有设备
- **说明：** 中速返回待机位置

### PARK_POS - 停靠位置
- **用途：** 设备停放点
- **默认值：** 位置=300, 速度=100
- **适用：** 机械臂、传送设备
- **说明：** 中速移动到停靠区

### MAINTENANCE_POS - 维护位置
- **用途：** 维修保养点
- **默认值：** 位置=500, 速度=50
- **适用：** 需要定期维护的设备
- **说明：** 低速移动到维护区域

## 🎓 最佳实践

### 1. 统一命名规范
```
✅ 推荐：
- HOME_POS, SAMPLE_POS, WASH_POS
- POSITION_1, POSITION_2, POSITION_3
- X_MIN, X_MAX, Y_MIN, Y_MAX

❌ 避免：
- pos1, pos2（不明确）
- 位置1, 位置2（中文可能导致问题）
- a, b, c（无意义）
```

### 2. 合理设置速度
```
原点/维护位置：50-80（低速安全）
常规工作位置：100-200（正常速度）
快速移动位置：200-500（高速移动）
```

### 3. 位置值范围
```
- 检查设备的实际运动范围
- 避免超出设备限位
- 留有安全余量
```

### 4. 批量添加顺序
```
推荐顺序：
1. 原点位置（HOME）
2. 主要工作位置
3. 辅助位置
4. 维护/停靠位置
```

## ⚠️ 注意事项

### 1. 设备选择
- 只显示支持位置的设备（电机、离心机、机器人）
- 泵类设备不在列表中（不支持位置）

### 2. 位置名称
- 必须填写，不能为空
- 建议使用英文和下划线
- 避免特殊字符

### 3. 数值验证
- 位置值：可以是负数
- 速度：必须大于0
- 都必须是有效的数字

### 4. 保存配置
- 添加位置后，记得点击"保存配置"
- 保存后才能永久生效
- 可以在右侧编辑器中继续调整

### 5. 连续添加模式
- 勾选后，每次添加会清空表单
- 设备选择保留，方便为同一设备添加多个位置
- 取消勾选或点击"取消"退出
