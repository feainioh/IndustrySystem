# PositionSettingsView - 快速参考卡

## 🚀 快速操作

### 添加设备 (3步)
```
1. 点击 [添加设备] 按钮（绿色）
2. 填写设备信息（设备类型、ID、名称）
3. 点击 [添加]
```

### 添加位置 (3步)
```
1. 点击 [添加位置] 按钮
2. 修改位置参数（名称、位置值、速度）
3. 点击 [保存配置]
```

### 删除位置 (2步)
```
1. 选择位置，点击 [删除] 按钮
2. 确认删除
```

## ⚡ 常用操作流程

### 为新项目配置设备
```
1. [添加设备] → 添加所需设备
2. [添加位置] → 为每个设备添加位置
3. [保存配置] → 保存并同步
4. [导出配置] → 备份配置文件
```

### 导入现有配置并修改
```
1. [导入] → 加载配置文件
2. [添加设备] → 添加新设备
3. [添加位置] → 补充位置
4. [保存配置] → 保存修改
```

### 清理无用位置
```
1. 选择位置
2. [删除] → 确认
3. [保存配置]
```

## 🔧 连接参数速查

### CAN 设备
- **CAN 节点 ID**: 1, 2, 3...

### 串口设备
- **串口**: COM1 ~ COM8
- **波特率**: 9600, 115200

### 网络设备
- **IP**: 192.168.1.xxx
- **端口**: 设备指定

### EtherCAT 设备
- **从站 ID**: 1, 2, 3...

## ⚠️ 重要提示

### ✅ 必须操作
- **设备 ID 必须唯一**
- **修改后必须保存**
- **验证连接参数**

### ❌ 不支持
- 直接删除设备（需手动编辑配置文件）
- 批量编辑位置（逐个修改）
- 设备参数详细配置（需编辑配置文件）

## 🚨 故障快速解决

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 无法添加设备 | 未导入配置 | 先导入配置 |
| 设备 ID 冲突 | ID 重复 | 更换唯一 ID |
| 位置未保存 | 忘记保存 | 点击保存配置 |
| 调试界面无设备 | 未同步 | 重新导入配置 |
