# 位置双向同步功能 - 快速使用指南

## ✅ 功能总览

现在 **设备调试界面** 和 **位置设置界面** 已实现完全双向同步！

### 支持的操作

| 操作 | 调试界面 → 位置设置 | 位置设置 → 调试界面 |
|------|-------------------|-------------------|
| 导入配置 | ✅ 自动同步 | ✅ 自动同步 |
| 添加位置 | - | ✅ 自动同步 |
| 修改位置 | - | ✅ 自动同步（保存后） |
| 删除位置 | - | ✅ 自动同步 |

## 📖 快速使用

### 1. 导入配置并同步

**在调试界面导入：**
```
DeviceDebugView → 导入配置 → 选择 JSON 文件
    ↓
✅ 自动同步到 PositionSettingsView
```

**在位置设置界面导入：**
```
PositionSettingsView → 导入配置 → 选择 JSON 文件
    ↓
（暂不自动同步到调试界面，需手动导入）
```

### 2. 添加新位置

**步骤：**
1. 打开 **PositionSettingsView**（位置设置）
2. 点击 **"添加位置"** 按钮
3. 系统自动创建新位置（默认使用第一个设备）
4. 修改位置参数：
   - 位置名称
   - 位置值
   - 速度
5. 点击 **"保存配置"**
6. ✅ 新位置自动同步到 DeviceDebugView

**效果：**
- 位置列表中出现新位置
- 调试界面可以使用该位置
- 统计信息自动更新

### 3. 修改位置参数

**步骤：**
1. 在 **PositionSettingsView** 选择要修改的位置
2. 修改位置值或速度
3. 位置自动标记为"已修改"
4. 点击 **"保存配置"**
5. ✅ 更改自动同步到 DeviceDebugView

**提示：** 修改后必须保存才能同步！

### 4. 删除位置

**步骤：**
1. 在 **PositionSettingsView** 选择要删除的位置
2. 点击 **"删除位置"** 按钮
3. 确认删除
4. ✅ 删除操作自动同步到 DeviceDebugView

### 5. 示教位置（调试界面）

**步骤：**
1. 在 **DeviceDebugView** 选择设备和位置
2. 点击 **"示教位置"** 按钮
3. 系统读取当前位置并保存
4. ⚠️ 暂不自动同步到 PositionSettingsView
5. 需要在调试界面保存配置后，重新加载才能同步

## 🎯 支持的设备类型

- ✅ **CAN 电机** - 完全支持
- ✅ **EtherCAT 电机** - 完全支持
- ✅ **离心机** - 完全支持
- ⚠️ **机器人** - 暂不支持（未来可扩展）

## ⚙️ 工作原理

### 同步机制

```
位置设置界面                         调试界面
┌─────────────┐                    ┌─────────────┐
│ 添加/修改/删除 │                    │  自动更新    │
│    位置     │  ──发布事件──>     │  对应设备    │
└─────────────┘                    └─────────────┘

调试界面                           位置设置界面
┌─────────────┐                    ┌─────────────┐
│  导入配置    │                    │  自动加载    │
│             │  ──发布事件──>     │  所有位置    │
└─────────────┘                    └─────────────┘
```

### 事件系统

使用 **Prism EventAggregator** 实现解耦：

- `DeviceConfigImportedEvent` - 配置导入事件
- `PositionUpdatedEvent` - 位置更新事件
- `PositionAddedEvent` - 位置添加事件
- `PositionDeletedEvent` - 位置删除事件

重复点击"添加位置" → 批量创建 → 统一修改参数 → 保存
```

### 2. 示教后同步
```
调试界面示教 → 保存配置 → 重新导入配置 → 同步完成
```

### 3. 验证同步状态
```
查看状态栏消息：
- "已从调试界面同步配置"

## ⚠️ 注意事项

### 1. 保存是必需的！
修改位置后，**必须点击"保存配置"才能同步到其他界面**。

### 2. 添加位置时选择设备
目前添加位置时自动使用第一个设备，未来可改为手动选择设备。

### 3. 调试界面示教暂不自动同步
调试界面的示教操作暂时不会自动同步到位置设置界面。
**解决方法：** 示教后保存配置，然后重新导入配置即可同步。

### 4. 删除位置需要确认
删除位置时会弹出确认对话框，确认后才会执行删除。

## 🔧 故障排除

### 问题：修改后没有同步
**原因：** 忘记点击"保存配置"  
**解决：** 修改后必须保存才能触发同步事件

### 问题：添加位置后列表没有更新
**原因：** 可能需要切换界面或刷新  
**解决：** 切换到其他界面再切回来，或重新导入配置

### 问题：删除位置后调试界面还显示
**原因：** 可能缓存了旧数据  
**解决：** 重新选择设备，或重启应用

### 问题：示教的位置没有同步
**原因：** 示教暂不自动同步  
**解决：** 示教后保存配置，然后重新导入
