# ?? Shell 界面现代化设计文档

## 设计理念

采用 **Fluent Design System** 和 **现代科技风格**，打造简洁流畅、富有科技感的工业系统界面。

## ?? 核心设计特点

### 1. 科技感深色主题
- **深色渐变背景**：从 `#0F1419` 到 `#1A2332` 的渐变，营造深邃的科技氛围
- **Accent 蓝色渐变**：`#0078D4` → `#00BCF2` 的渐变，增强视觉焦点
- **发光效果**：关键元素添加蓝色发光效果，提升科技感

### 2. 自定义标题栏
```xaml
<Grid Height="48" Background="{StaticResource TechGradientBrush}">
```

**特点**：
- ? 48px 高度，现代简洁
- ?? 渐变背景 + 阴影效果
- ?? Logo 带发光效果
- ?? 集成用户信息和主题切换
- ?? 快速登出按钮

### 3. 彩色导航图标
每个功能模块使用不同颜色，增强视觉识别：

| 模块 | 颜色 | 色值 | 含义 |
|------|------|------|------|
| 运行看板 | 亮蓝 | `#00BCF2` | 活跃、实时 |
| 实验管理 | 绿色 | `#10893E` | 科学、实验 |
| 物料管理 | 橙色 | `#FF8C00` | 物资、库存 |
| 数据管理 | 紫色 | `#8764B8` | 数据、分析 |
| 设备管理 | 红色 | `#E74856` | 设备、硬件 |
| 系统管理 | 黄色 | `#FFB900` | 设置、管理 |
| 告警监控 | 警示红 | `#FF4444` | 警告、注意 |

### 4. 卡片式内容区域
```xaml
<Border CornerRadius="8" 
        Margin="8"
        Background="..."
        Effect="DropShadowEffect">
```

**特点**：
- ?? 圆角 8px，现代柔和
- ?? 轻微阴影，层次分明
- ?? 适当边距，视觉舒适

### 5. 流畅动画
```xaml
<Style x:Key="FadeInAnimation">
    <DoubleAnimation From="0" To="1" Duration="0:0:0.5"/>
</Style>
```

**效果**：
- 淡入效果，视觉流畅
- 0.5秒过渡，不快不慢
- 减速曲线，自然舒适

**用途**：标题栏、导航栏背景

## ?? 使用说明

### 修改颜色
在 `Shell.xaml` 的 `<Window.Resources>` 中修改：
```xaml
<LinearGradientBrush x:Key="TechGradientBrush">
    <GradientStop Color="你的颜色" Offset="0"/>
</LinearGradientBrush>
```

### 添加新菜单
```xaml
<ui:NavigationViewItem Content="新功能" Tag="NewFeature">
    <ui:NavigationViewItem.Icon>
        <ui:FontIcon Glyph="&#xE12B;" Foreground="#00BCF2"/>
    </ui:NavigationViewItem.Icon>
</ui:NavigationViewItem>
```

### 修改动画速度
```xaml
<DoubleAnimation Duration="0:0:1"/>  <!-- 1秒 -->
```
